The Homelab Project

Introduzione

L’homelabbing è la pratica di creare un piccolo laboratorio informatico domestico, dove poter sperimentare, imparare e gestire servizi personali in autonomia.
Può variare da semplici configurazioni con un singolo Raspberry Pi fino a vere e proprie infrastrutture con più server, reti virtuali e sistemi di automazione.
L’obiettivo è avere pieno controllo dei propri dati e servizi, migliorando le proprie competenze tecniche e riducendo la dipendenza da soluzioni cloud esterne.

Configurazione

La mia configurazione è strutturata in modo modulare e sostenibile. Alla base c’è pfSense come router principale, posizionato a monte della rete. In caso di fibra o ADSL, il modem funge da ponte verso pfSense, che gestisce routing, firewall e la VPN WireGuard per l’accesso remoto sicuro. Dietro al router si trovano lo switch e gli access point per distribuire la connessione in tutta la rete domestica. Poi arriva la parte “intelligente” dell’infrastruttura: Server Core È il cuore del sistema, ospitato su una macchina a basso consumo (come un Raspberry Pi, una Zimaboard o un vecchio portatile). Su questo server girano i servizi che devono essere sempre attivi: Home Assistant per l’automazione domestica un server web con script per accensione, spegnimento e gestione remota degli altri server Server Home È il “contenitore” principale dei dati, simile a un NAS. Rimane spento la maggior parte del tempo e viene acceso solo quando serve accedere a grandi quantità di file o a risorse più pesanti, riducendo così i consumi. Server AI (opzionale) Pensato per chi vuole sperimentare con intelligenza artificiale, machine learning o modelli locali. Ovviamente richiede hardware più costoso e maggiore potenza di calcolo. Server di Backup Serve a garantire la sicurezza dei dati, mantenendo copie aggiornate in locale o su dispositivi separati.

Conclusioni

Costi principali:

  • Energia elettrica
  • Connessione Internet
  • Manutenzione hardware e software

Benefici:

  • Maggiore privacy e controllo sui propri dati
  • Nessun costo ricorrente per servizi cloud
  • Indipendenza e libertà di sperimentare
